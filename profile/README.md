## 🛠 App Support / 문의

### 📌 앱 설명
BlockList는 블록체인 상에서 거래되는 스테이블 코인 지갑 주소 위험도를 AI 기반으로 분석하는 서비스입니다.  
GNN 기반 위험도 예측, Neo4j 거래 그래프 분석, 랭체인 기반 자연어 해석을 통해 지갑주소의 위험 점수와 근거를 사용자에게 제공합니다.

BlockList는 거래소·핀테크·PG사가 사용하는 B2B 온체인 위험 분석 서비스이지만, 그 목적은 기업 편의가 아니라 일반 사용자의 일상 금융 안전을 지키는 것에 있습니다.
스테이블 코인 결제와 디지털 자산 사용이 일상화되는 환경에서, 사용자는 거래 상대의 위험성을 직접 판단하기 어렵습니다. BlockList는 이러한 불편과 불안을 해소하기 위해, 위험 지갑을 사전에 차단하고 AI가 “왜 위험한지”를 이해하기 쉬운 언어로 설명하도록 설계되었습니다.
기업 서비스 내부에 자연스럽게 스며들어 사용자 경험을 방해하지 않으면서도,
뒤에서 조용히 일상 속 디지털 금융을 더 안전하게 만들어주는 생활밀착형 AI 안전장치가 되는 것이 BlockList의 핵심 목표입니다.

서브스의 목적과 의의, 시연 방법 등은 성과보고서와 위의 내용을 참고해주시고, AI와 iOS 개발의 기술적인 부분이 궁금하시다면 하단의 Readme를 참고해주세요.
